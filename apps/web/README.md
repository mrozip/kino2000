# kino2000 Web

React, TypeScript, and Vite placeholder front end for sprint 0.

The Sprint 0 page is intentionally minimal and currently renders the placeholder welcome screen. Shared root commands should be preferred for full validation.

## Commands

```sh
npm run dev
npm run test --workspace @kino2000/web
npm run typecheck --workspace @kino2000/web
npm run build --workspace @kino2000/web
```

Equivalent workspace-only commands:

```sh
npm run dev --workspace @kino2000/web
npm run test --workspace @kino2000/web
npm run build --workspace @kino2000/web
```
