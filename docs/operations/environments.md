# Environments

## Local

Used for:

- Application development
- Unit testing
- Mocked API responses
- Local Azure Functions execution

## Development

A shared Azure environment automatically deployed from the main branch.

Used for:

- Integration testing
- Authentication testing
- Cosmos DB testing
- Deployment validation
- Demonstrations

## Production

Production should be introduced only after the core movie-tracking functionality is stable.

Production deployment requirements are described in [deployment](deployment.md).
