# Sprint 6 - Reviews and Interaction

## Goal

Allow users to write and interact with movie reviews.

## Features

- Write reviews
- Edit own reviews
- Delete own reviews
- Display reviews on movie pages
- Like reviews
- Report inappropriate content
- Add review length limits
- Add basic moderation workflows
- Add rate limiting

Direct messaging and unrestricted comments should remain outside the initial scope until blocking, moderation, privacy, and abuse-handling requirements have been designed.
