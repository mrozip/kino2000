# Sprint 4 - Public Profiles

## Goal

Allow users to create a public identity and share selected movie activity.

## Features

- Add unique usernames
- Add public profile routes
- Add editable profile bio
- Display public favourites
- Display public ratings
- Add privacy controls
- Prevent exposure of Entra identity identifiers
- Handle unavailable usernames
- Handle username changes

## Example Route

```text
/users/:username
```
